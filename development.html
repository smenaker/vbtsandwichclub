Developing this App

I've never made a website or used Python this summer, so developing this app was a real experience in expanding my knowledge. It's also a testament to how easy Google, Python, and Django make web app development. The environment encourages experimentation and allows you to test with immediate feedback.

If you'd like to contribute, you can checkout the code from http://vbtsandwichclub.googlecode.com/svn/trunk/ and poke around it. If you have any particular patches or would like to become more involved, email kevinl@voicebox.com. I found Google's Getting Started With App Engine and O'Reilly's similarly named Getting Started With Google's App Engine to be extremely helpful in creating this webapp (you'll notice I lifted the CSS from O'Reilly's tutorial).
